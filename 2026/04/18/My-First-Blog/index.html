v1.0.3 | 2026-04-19 下午

• 资源路径重构：彻底解决 MarkText 自动转义绝对路径导致的“图裂”问题，统一回归 /images/ 根目录逻辑。

• 部署权限修复：重置 ED25519 认证密钥并完成 GitHub 身份对等验证，解决 Permission denied 报错。

• 自动化部署博弈：在 GitHub Action 秘钥死锁与本地手动部署之间达成平衡，确保源码与静态网页双端同步。
